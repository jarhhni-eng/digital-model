# Beery VMI shape images

Place the official Beery VMI geometric shape images here.

**Required files:** `shape_1.png` through `shape_27.png` (or your test length).

- `shape_1.png` – Item 1 model shape  
- `shape_2.png` – Item 2 model shape  
- …  
- `shape_27.png` – Item 27 model shape  

The app loads them from `/beery-vmi/shape_N.png`. If an image is missing, the test page shows a placeholder for that item.
