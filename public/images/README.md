# Test images

## WAIS Visual Puzzles (Visuo-Constructive)

Place the puzzle images for the Visuo-Constructive test in this folder.

**Required files:**  
`constructive_page-0003.jpg` through `constructive_page-0027.jpg`

- Each image contains the main puzzle and the four answer options (A, B, C, D).
- Question 1 → `constructive_page-0003.jpg`  
- Question 2 → `constructive_page-0004.jpg`  
- …  
- Question 25 → `constructive_page-0027.jpg`

URLs in the app: `/images/constructive_page-XXXX.jpg`

## TVPS-3 (Visuo-Perceptive)

Place the TVPS-3 item images in this folder.

**Required files:**  
`TVPS_page-0001.jpg` through `TVPS_page-0112.jpg`

- 7 subtests × 16 items = 112 items. Each image shows the stimulus and answer options.
- Item 1 → `TVPS_page-0001.jpg`, Item 2 → `TVPS_page-0002.jpg`, … Item 112 → `TVPS_page-0112.jpg`

URLs in the app: `/images/TVPS_page-XXXX.jpg`
