# 2D Mental Rotation — image assets

Place 19 images here, named exactly:

```
Rotation (1).png
Rotation (2).png
…
Rotation (19).png
```

Each image must contain:

- a **reference figure** at the top, and
- two **candidate figures** below — one on the **left**, one on the **right**.

The user picks the side (LEFT or RIGHT) that matches the reference after a 2D rotation.

Wired in `lib/mental-rotation-2d-test.ts` (path pattern: `/rotation-2d/Rotation (N).png`).
